# ZHITWeb
镇海信息技术协会网站

简介:
---------------
**项目搭建:**

  * 该项目包括前台界面的展示和管理端的实现。
  * 使用 Java + sqlserver 完成后台逻辑的实现；
  * 管理端采用 easyui 作为框架；
  * 使用 jQuery 完成网站前端 JS 脚本和样式处理;
